NaukriMitra Platform

Role: Full Stack & API Developer

A recruitment platform designed to enable real-time job tracking, location-based matching, and seamless communication between recruiters and candidates.

Problem

Recruitment systems lacked real-time tracking, centralized dashboards, and efficient location-based job matching. This resulted in delays, reduced hiring efficiency, and poor user experience.

Solution

Developed a scalable API-driven job platform with real-time communication features and geolocation-based job discovery, improving hiring workflows and user engagement.

Architecture

  • Admin & HR dashboards for centralized control
  • API-based job management system
  • Google Maps integration for location-based job matching
  • Phone communication APIs for real-time interaction

My Contribution

  • Developed Admin and HR dashboards
  • Built APIs for job posting, tracking, and management
  • Integrated Google Maps API for geo-based job discovery
  • Integrated phone communication APIs
  • Optimized backend performance for real-time updates

Challenges

  • Implementing real-time job tracking across multiple users
  • Handling accurate and scalable location-based services
  • Maintaining performance with growing job and user data

Impact

  • Improved hiring efficiency for recruiters
  • Enhanced experience for job seekers
  • Enabled real-time job updates and tracking
  • Reduced manual recruitment processes

Tech Stack

Node.js Angular MongoDB Express.js Google Maps API
