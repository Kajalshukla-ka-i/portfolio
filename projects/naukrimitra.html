NaukriMitra Platform

Problem

Recruitment processes lacked real-time tracking, centralized dashboards, and efficient location-based job matching. This resulted in delays, poor coordination between recruiters and candidates, and reduced hiring efficiency.

Solution

Developed a scalable job platform using an API-driven architecture that enables real-time job tracking, location-based matching, and seamless communication between recruiters and job seekers.

Architecture

  • Admin & HR dashboards for centralized control
  • API-based backend for job management
  • Google Maps integration for location-based job discovery
  • Phone communication APIs for real-time interaction

My Contribution

  • Developed Admin and HR dashboards
  • Built APIs for job posting, tracking, and management
  • Integrated Google Maps API for geo-based job search
  • Integrated phone communication APIs for real-time connectivity

Challenges

  • Implementing real-time job tracking across multiple users
  • Handling accurate location-based services
  • Maintaining performance with growing user and job data

Impact

  • Improved hiring efficiency for recruiters
  • Enhanced user experience for job seekers
  • Enabled real-time job updates and tracking

Tech Stack

Node.js, Angular, MongoDB, Express.js, Google Maps API
